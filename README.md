## Installation

```bash
pip install -U media-filesize-estimator
```

or install with `Poetry`

```bash
poetry add media-filesize-estimator
```

Then you can run

```bash
media-filesize-estimator --help
```

or with `Poetry`:

```bash
poetry run media-filesize-estimator --help
```
